Azure Certification Paths for SQL Server DBAs

Azure Fundamentals

  • AZ-900: Microsoft Azure Fundamentals - Learn more

    This certification covers foundational knowledge of cloud services and how those services are provided with Microsoft Azure.

  • DP-900: Microsoft Azure Data Fundamentals - Learn more

    Focuses on core data concepts and how they are implemented using Microsoft Azure data services.

  • AI-900: Microsoft Azure AI Fundamentals - Learn more

    Provides an understanding of AI and machine learning workloads and considerations on Azure.

  • SC-900: Microsoft Security, Compliance, and Identity Fundamentals - Learn more

    Covers security, compliance, and identity concepts and related cloud-based Microsoft solutions.

Core Azure Services

  • AZ-104: Microsoft Azure Administrator - Learn more

    Manages cloud services that span storage, networking, and compute cloud capabilities, with a deep understanding of each service across the full IT lifecycle.

  • AZ-305: Designing Microsoft Azure Infrastructure Solutions - Learn more

    Involved in advising stakeholders and translating business requirements into secure, scalable, and reliable solutions.

  • AZ-400: Designing and Implementing Microsoft DevOps Solutions - Learn more

    Requires candidates to design and implement strategies for application code and infrastructure that allow for continuous integration, testing, delivery, monitoring, and feedback.

Data & AI

  • PL-300: Microsoft Power BI Data Analyst - Learn more

    Empowers candidates to become proficient in developing actionable insights by leveraging Power BI to analyze data.

  • DP-300: Administering Relational Databases on Microsoft Azure - Learn more

    Targets professionals managing and overseeing database solutions in cloud environments, monitoring operational aspects, and optimizing performance.

  • DP-203: Data Engineering on Microsoft Azure - Learn more

    Designed for candidates looking to design and implement the management, monitoring, security, and privacy of data using the full stack of Azure data services.
